# Open Systems Interconnection (OSI)

| Layer | Name         | Addressing                 | Protocol |
| ----- | ------------ | -------------------------- | -------- |
| 1     | Physical     |                            |          |
| 2     | Data Link    | Media Access Control (MAC) | ARP      |
| 3     | Network      | Internet Protocol (IP)     | DHCP     |
| 4     | Transport    | Port number                | TCP/UDP  |
| 5     | Session      |                            |          |
| 6     | Presentation |                            |          |
| 7     | Application  |                            |          |
