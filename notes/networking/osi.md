| ----- | ------------ | -------------------------- |
| 1     | Physical     |                            |
| 2     | Data Link    | Media Access Control (MAC) |
| 3     | Network      | Internet Protocol (IP)     |
| 4     | Transport    | Port number                |
| 5     | Session      |                            |
| 6     | Presentation |                            |
| 7     | Application  |                            |
